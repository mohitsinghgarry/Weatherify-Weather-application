### Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mohitsinghgarry/Weatherify-Weather-application.git
   cd Weatherify-Weather-application

   ```bash
   npm install
   ```

3. **Set up environment variables**:
